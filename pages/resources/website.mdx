If you want to notch it up and explore more templates, you can use any template online and edit it accordingly. Here are some of the recommended websites:

- [Bootstrap templates](http://getbootstrap.com/)

### Basic steps to use a website’s template:

1.	Save the webpage
2.	Open the “index.html” file with a text editor 
3.	Edit accordingly by replacing the text within the HTML tags with your information
